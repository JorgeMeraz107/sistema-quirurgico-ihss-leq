Sistema Quirúrgico LEQ - IHSS

Sistema de Gestión de Operaciones Quirúrgicas desarrollado para el Instituto Hondureño de Seguridad Social (IHSS).

📌 Descripción

LEQ es un sistema de escritorio desarrollado como reemplazo del antiguo sistema quirúrgico institucional construido en FoxPro hace aproximadamente 20 años por una consultora internacional.

El sistema anterior únicamente funcionaba sobre Windows XP y debía ejecutarse mediante máquinas virtuales usando VirtualBox, lo cual generaba limitaciones técnicas, problemas de mantenimiento y dependencia de software obsoleto.

La nueva versión fue desarrollada completamente en Java, modernizando la infraestructura del sistema y permitiendo su ejecución en múltiples sistemas operativos.

🚀 Tecnologías Utilizadas
Java SE
Swing / Java Desktop UI
JDBC
MySQL
JasperReports
Apache POI (Excel / Word)
HTML Reports
Scripts Bash para automatización
Encriptación y hashing de contraseñas
🔐 Características Principales
Gestión completa de operaciones quirúrgicas
Administración de pacientes
Sistema de usuarios y roles
Permisos y habilitaciones administrativas
Contraseñas hasheadas y usuarios encriptados
Dashboards y estadísticas gráficas
Generación de reportes
Exportación a Excel, Word y PDF
Sistema multiplataforma
Backups automáticos diarios y semanales
Automatización mediante scripts Bash en servidores del Data Center
